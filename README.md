# ✈️ AI Travel Planner

## Overview
AI Travel Planner is a web application that helps users plan trips by collecting travel preferences and generating trip details. It provides a simple booking flow with confirmation and booking history views.  
This project is developed as a college mini / final year project.

---

## Features
- Travel planning based on user inputs  
- Booking confirmation  
- Booking history and trip details  
- Clean and user-friendly interface  

---

## Tech Stack
**Frontend:** HTML, CSS, JavaScript  
**Backend:** Python, Flask / FastAPI  
**Database:** SQLite / PostgreSQL  

---

## Workflow
1. User enters travel details  
2. Backend processes the request  
3. Trip plan is generated  
4. Booking details are stored  
5. Booking history is displayed
